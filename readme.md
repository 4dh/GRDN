## Setup
- setup conda environment 
  >*conda create --name=GRDN_env*
- install dependencies
  >*pip install -r requirements.txt*

## Running App
- navigate to ...GRDN/src
- activate environment
  >*conda activate GRDN_env*
- run app
  >*python -m streamlit run app.py*
